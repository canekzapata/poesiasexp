nefelógrafo :: editor — poesiasexp

el poema llega de python (nefelografo.py --web), por fetch
de poema.json, por [abrir] o arrastrando un .json aquí.

gestos
  arrastrar      mover la palabra
  clic           mutar: la palabra gira por sus variantes
                 (familia, antepasados, náhuatl, deriva sonora)
  doble clic     editar el texto a mano
  rueda          crecer / encoger

teclas (con una palabra elegida)
  r / R          girar +15° / −15°
  + / -          crecer / encoger
  x / supr       borrar
  esc            soltar

teclas (siempre)
  a              animar          e    esquizo
  g              rejilla         t    tema noche/papel
  v              variar (temblor)

restricciones
el ticker vigila: máximo 12 palabras-base, mínimo 30% de vacío.
romperlas es tu derecho: el instrumento solo lo hace visible.

clic en cualquier parte para volver.
